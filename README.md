# 2018

Evento LinguÁgil 2018 - Salvador/BA

## Como iniciar o site

Certifique-se de você tem o Docker instalado

    $ git clone git@github.com:linguagil/2018.git
    $ Cd 2018
    $ make serve

Abra a url: http://0.0.0.0:4000/
